Доска задач

Учебный проект на Spring Boot

О проекте

Это учебный проект, демонстрирующий использование различных технологий Spring, таких как Spring Security, Hibernate, WebMVC, OpenAPI документация и аспектно-ориентированное программирование. Приложение представляет собой менеджер задач в виде Scrum/Kanban доски.

Используемые технологии

Spring Boot Spring Security Hibernate Spring WebMVC OpenAPI/Swagger AOP Thymeleaf

Основные функции

Управление задачами

Создавайте и назначайте задачи другим пользователям. Просматривайте назначенные вам задачи или созданные вами.

Командная доска

Просматривайте общую доску команды и статусы задач. Отслеживайте прогресс работы всей команды.

Комментарии

Оставляйте комментарии к конкретным задачам. Обсуждайте детали выполнения с коллегами.

API документация

Доступ к Swagger UI по адресу: /swagger-ui/index.html#/. Изучайте доступные API endpoints.

Настройки пользователя

Изменяйте свои учетные данные и пароль. Персонализируйте свой профиль.

Логи (для админов)

Администраторы могут просматривать логи системы и скачивать их в формате JSON.
